CRS Data

This application stores CRS data so you can lookup the three letter code of rail stations.

These codes are standard throughout TOC's

The list rarely changes, but it is best to reload the data after every couple of months, or when you first use this application.

Simply press the update button and wait till the date on the page changes to todays date.

You can then go back to the count or journey page you were on, or just exit the application if you have no counts to submit.
