General Help


- Use the buttons on the top frame of the app to navigate through the screen. Larger screen phones and tablets will have easy navigate buttons on each screen in prominent positions.

- Make sure to enter your personal information before submitting headcounts. Do this using the home screen My Settinga button, or by clicking the settings buttons one the home page (top left) and clicking My Settings

- Please use CRS codes for station names. Live CRS data is available on the relevant fields when you start to type in a station name.
